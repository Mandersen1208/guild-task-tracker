# guild-task-tracker

Small responsive React task tracker with local persistence.

**Legitimate React application** — Vite + TypeScript + Tailwind.

## Features (strictly limited)
- Create tasks
- Mark complete
- Filter (All / Active / Completed)
- Delete tasks
- Persists to localStorage
